A Dark Kingdom

Welcome to A Dark QKingdom, a real-time kingdom simulator merging Pathfinder: Kingmaker rules and Dwarf Fortress pop simulation, built with a "Dark Room" survival prologue.

Quick Start for Agents & Developers

This is a dual-track project. Mechanical parity is strictly enforced between the Python backend and the React frontend. When testing changes, you must run and verify both environments:

1. Python CLI Engine

The terminal-based simulation built with rich.

cd engine
python Engine.py


To run the unit tests:

python test_engine.py


2. React/Vite Web App

The visual dashboard frontend built with React and Tailwind CSS.

cd frontend
pnpm install
pnpm run dev
